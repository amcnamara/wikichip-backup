Arrandale Core[edit]

The Arrandale-based Pentium processors are dual-core processors. These processors had the FSB replaced with Direct Media Interface 1.0. These processors use Socket-G1. They all have the following features:

  • Bus: DMI (2.5 GT/s) interface
  • TDP: 18, 35 W
  • ISA: Everything up to SSSE3 (SMM, FPU, NX, MMX, SSE, SSE2, SSE3, SSSE3).
  • Tech: SpeedStep (EIST), FMA, Flex Memory

Sandy Bridge Âµarch[edit]

Main article: Sandy Bridge Microarchitecture

Mobile[edit]

See also: Sandy Bridge M

Manufactured on a 32 nm process, mobile Sandy Bridge-based budget Pentium processors were introduced in mid-2011. Mobile Pentium uses FCBGA-1023 packages and rPGA988B for B models. Sandy Bridge incorporated the integrated graphics on this same die as the newly architected CPU, allowing for higher performance and lower power than previous generation. Additionally, those models added SSE4.1 and SSE4.2 support on mobile Pentium models for the first time.

  • TDP: 17 W, 35 W
  • ISA: Everything up to SSE4.2 (SMM, FPU, NX, MMX, SSE, SSE2, SSE3, SSSE3, SSE4.1, SSE4.2)
  • Tech: SpeedStep (EIST), FMA, Flex Memory
  • GPU: HD Graphics @ 350/650 MHz with bursts of 800-1,150 MHz

Skylake Âµarch[edit]

Main article: Skylake Microarchitecture

Desktop[edit]

See also: Skylake S

Intel released the desktop Skylake-based budget processors in late 2015. All desktop models utilize the LGA-1151 socket which is used for most of the Skylake desktop lineup. Skylake-based Pentium models received a large set of improvements over their haswell counterparts, including a higher performing core, higher rate chipset and PCIe lanes which were upgraded to Gen3, and faster DDR4 memory.

  • TDP: 35 W, 51 W, and 54 W
  • ISA: Everything up to SSE4.2 (SMM, FPU, NX, MMX, SSE, SSE2, SSE3, SSSE3, SSE4.1, SSE4.2, AES)
  • Tech: VT-x/VT-d, EPT, SpeedStep (EIST), IPT, Secure Key, Software Guard (SGX).
  • GPU: HD Graphics 530/510 @ 350 MHz with bursts of 0.95-1.05 GHz
  • Mem: Up to DDR4-2133, DDR3L-1600 ECC

Mobile[edit]

See also: Skylake U and Skylake Y

Mobile Skylake-based budget Pentium processors were introduced in late 2015. Mobile Pentium uses FCLGA-1515/FCLGA-1356 sockets which incorporate the chipset as well as the microprocessor itself in the same package. Skylake-based mobile Pentium processors inherit most of the benefits from the new architecture, including a faster core and faster memory support.

  • TDP: 6 W - 15 W (With a Configurable TDP-down of 4.5 - 10 W)
  • ISA: Everything up to SSE4.2 (SMM, FPU, NX, MMX, SSE, SSE2, SSE3, SSSE3, SSE4.1, SSE4.2, and AES)
  • Tech: Hyper-Threading, VT-x, VT-d/EPT, SpeedStep (EIST), Flex Memory, IPT, Smart Response Technology, My WiFi Technology, Secure Key, Software Guard (SGX)
  • GPU: HD Graphics 510 @ 300 MHz with bursts of 800-950 MHz

Kaby Lake Âµarch[edit]

Main article: Kaby Lake Microarchitecture

Desktop[edit]

See also: Kaby Lake S

Desktop Kaby Lake-based budget Pentium processors were introduced in early 2017. In addition to enjoying a modest performance increase due to their slightly higher clock frequency, those processors feature hyper-threading support for the first time as well as introduced a number of new security features including Memory Protection (MPX) and OS Guard. Kaby Lake-based Pentiums utilize the same Socket LGA-1151 as Skylake. All models share the following common features:

  • TDP: 35 W and 51 W
  • ISA: Everything up to SSE4.2 (SMM, FPU, NX, MMX, SSE, SSE2, SSE3, SSSE3, SSE4.1, SSE4.2, AES)
  • Tech: Hyper-Threading, VT-x/VT-d, EPT, SpeedStep (EIST), IPT, Secure Key, Software Guard (SGX), and Memory Protection (MPX), and OS Guard.
  • GPU: HD Graphics 630 @ 350 MHz with bursts of 1-1.05 GHz
  • Mem: Up to DDR4-2400, DDR3L-1600 ECC

Mobile Kaby Lake-based budget Pentium processors were introduced in early 2017. Mobile Pentium uses FCLGA-1515/FCLGA-1356 sockets which incorporate the chipset as well as the microprocessor itself in the same package. In addition to enjoying a modest performance increase due to their slightly higher clock frequency, a number of new features were enabled on Pentium processors for the first time including Memory Protection (MPX) and OS Guard. All models share the following common features:

  • TDP: 4.5 W - 15 W (With a Configurable TDP-down of 4.5 - 10 W)
  • ISA: Everything up to SSE4.2 (SMM, FPU, NX, MMX, SSE, SSE2, SSE3, SSSE3, SSE4.1, SSE4.2, and AES)
  • Tech: Hyper-Threading, VT-x, VT-d/EPT, SpeedStep (EIST), Flex Memory, IPT, Smart Response Technology, My WiFi Technology, Secure Key, Software Guard (SGX), Memory Protection (MPX), and OS Guard
  • GPU: HD Graphics 610 @ 350 MHz with bursts of 1-1.05 GHz
