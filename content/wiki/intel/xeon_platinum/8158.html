Xeon Platinum 8158 is a 64-bit x86 high-performance server multiprocessor set to be introduced by Intel in the second quarter of 2017. This processor is based on the server configuration of the Skylake microarchitecture (a Skylake SP core) and is manufactured on Intel's 14 nm process. The 8158 operates at 3.0 GHz.
