Am186EM–33VC\W is a microcontroller designed by AMD and introduced in 1995. This MCU incorporated the Am186 core which operated at 33 MHz. Additionally this chip included many other features such as 32 general purpose I/O lines, DMA controller, UART, and a watchdog timer.

Graphics[edit]

This chip had no integrated graphics processing unit.

Features[edit]

  • Single clock input
  • Interrupt Control Unit
  • General Purpose DMA (2 independent channels)
  • 3 x 16-bit timers
  • Watchdog Timer
  • UART
  • 32x General Purpose IO
  • Synchronous Serial Interface
  • 6 x external interrupts
