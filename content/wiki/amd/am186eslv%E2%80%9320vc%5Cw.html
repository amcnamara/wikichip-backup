Am186ESLV–20VC\W is a microcontroller designed by AMD. This MCU incorporated the Am186 core which operated at 20 MHz. Additionally this chip included many other features such as 32 general purpose I/O lines, DMA controller, UART, and a watchdog timer. As with the rest of the Am186ES series, this chip included a serial port hardware handshaking and baud rate generator as well as more extensive set of internal and external interrupt line. This model is a low-voltage version of the Am186ES–20VC\W.

Graphics[edit]

This chip had no integrated graphics processing unit.

Features[edit]

  • Low voltage
  • Single clock input
  • Interrupt Control Unit
  • General Purpose DMA (2 independent channels)
  • 3 x 16-bit timers
  • Watchdog Timer
  • UART
  • 32x General Purpose IO
  • Synchronous Serial Interface
  • 8-bit or 16-bit memory and I/O static bus modes
  • Serial port hardware handshaking
  • independent serial port baud rate generators
  • Watchdog timer can generate NMI or reset
  • Pseudo static RAM (PSRAM) controller
  • pulse-width demodulation
  • 8 x external interrupts
  • 8 x internal interrupts
