Am186CH-40KI\W is a 16-bit HDLC communication microcontroller designed by AMD. This MCU implements the Am186 core operating at 40 MHz and incorporates a number of High-level Data Link Controls along with 4 dedicated DMA channels for it. This model has industrial temperature range support.

Graphics[edit]

This chip had no integrated graphics processing unit.

Features[edit]

  • Industrial temperture range
  • High-Speed UART with autobaud
  • Synchronous serial interface (SSI)
  • 3x programmable 16-bit timers
  • Hardware watchdog timer
  • 4x General-purpose DMA channels
  • 48x Programmable I/O ports
  • 36x maskable interrupts
  • Interrupt Controller
  • Integrated DRAM controller
  • 14x chip selects (8x peripherals, 6x memory)
  • Multiplexed and nonmultiplexed address/data bus
  • 2x High-level Data Link Control (HDLC) channels
  • 2x Time Slot Assigners (TSAs)
  • 4x dedicated DMA channels (4x HDLC)
